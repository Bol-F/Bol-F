# Publishing this GitHub profile

1. Review the personal details in `README.md`.
2. Optionally add a LinkedIn link beside the existing badges.
3. Create a **public** GitHub repository named exactly `Bol-F` under the `Bol-F` account.
4. Copy the contents of this folder to the repository root, then commit and push them.
5. Open **Actions → Update profile README → Run workflow** once.

The workflow refreshes the Featured Projects table every Monday. It selects up to four public repositories owned by `Bol-F`, excluding forks, archived repositories, and the profile repository itself.
